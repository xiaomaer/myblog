1、Sass简介

Sass是一种"CSS预处理器"（css preprocessor）。它的基本思想是：用一种专门的编程语言，进行网页样式设计，然后再编译成正常的CSS文件。（注意：CSS不是一种编程语言）

Sass 是对CSS的扩展，让CSS语言更强大、优雅。它允许你使用变量、嵌套规则、mixins、导入等众多功能，并且完全兼容CSS语法。 Sass有助于保持大型样式表结构良好，同时也让你能够快速开始小型项目，特别是在搭配Compass样式库一同使用时。

特色

  • 完全兼容 CSS3

  • 在 CSS 语言基础上添加了扩展功能，比如变量、嵌套 (nesting)、混合 (mixin)

  • 对颜色和其它值进行操作的{Sass::Script::Functions 函数}

  • 函数库控制指令之类的高级功能

  • 良好的格式，可对输出格式进行定制

  • 支持 Firebug

3、编译

SASS文件就是普通的文本文件，sass有两种后缀名文件：一种后缀名为sass，不使用大括号和分号；另一种就是我们这里使用的scss文件，该文件可以直接使用CSS语法，使用大括号和分号。通常都使用.scss后缀，本文所有的示例都是.scss文件。在此也建议使用后缀名为scss的文件，以避免sass后缀名的严格格式要求报错。

两者形式上的差异，请见下面代码：

以下这段简单的 Sass 代码： 

#sidebar
  width: 30%
  background-color: #faa

只需添加花括号和分号就能转换为 SCSS 语法： 

#sidebar {
  width: 30%;
  background-color: #faa;
}

另外，SCSS 对空白符号不敏感。上面的代码也可以书写成下面的样子： 

#sidebar {width: 30%; background-color: #faa}

①在ruby命令行界面输入下面的命令，可以查看.scss文件编译后的css代码，命令如下: 

sass learn_sass.scss

执行命令的结果如下图所示：

②在ruby命令行界面输入下面的命令，将把.scss文件编译后的结果保存为.css文件，命令如下： 

sass learn_sass.scss test.css

执行该命令后，在同一目录下将会自动生成test.css文件，结果如下图所示：

③在ruby命令行输入如下命令，可以让sass自动监视某个文件或目录的变化并做相应的处理，如：发现.scss文件变化，显示提示信息；发现.scss文件变化，自动生成编译后的css文件。

// 监视.scss文件的变化，并自动生成css文件

// 监视一个目录的变化
sass --watch css3_animals/css

④SASS四种编译风格 

* nested：嵌套缩进的css代码，它是默认值。
* expanded：没有缩进的、扩展的css代码。
* compact：简洁格式的css代码。
* compressed：压缩后的css代码。

开发环境下，通常使用默认编译风格；而生产环境下，一般使用最后一种风格，命令行如下： 

sass --style compressed test.sass test.css

四种风格编译后的CSS代码样式如下： 

// nested
#main {
  color: #fff;
  background-color: #000; }
  #main p {
    width: 10em; }

.huge {
  font-size: 10em;
  font-weight: bold;
  text-decoration: underline; }

// expanded
#main {
  color: #fff;
  background-color: #000;
}
#main p {
  width: 10em;
}

.huge {
  font-size: 10em;
  font-weight: bold;
  text-decoration: underline;
}

// compact
#main { color: #fff; background-color: #000; }
#main p { width: 10em; }

.huge { font-size: 10em; font-weight: bold; text-decoration: underline; }

// compressed
#main{color:#fff;background-color:#000}#main p{width:10em}.huge{font-size:10em;font-weight:bold;text-decoration:underline}

⑤使用 sass --help 可以列出完整的帮助文档。

4、基本用法

Sass的官网，提供了一个在线转换器（http://www.sassmeister.com/）。在学习语法的过程中，您可以在哪里编辑和运行下面的示例。

①变量

sass可以定义变量，sass的变量必须以$开头，并且后面紧跟着变量名，而变量名和变量值之间使用冒号(:)分割，像定义css属性一样。如果值后面加上!default则表示默认值。

a.变量声明

sass变量的声明和css属性的声明很像，格式为“$变量名:变量值”，代码如下： 

$primary-color:blue;

以上代码表示变量$primary-color现在的值为blue。任何CSS属性值都可以作为sass变量值，甚至是以空格分隔的多个属性值，或者是以逗号分隔的多个属性值，代码如下： 

$basic-border: 1px solid black;
plain-font: Arial,sans-serif,sans-serif;

b.变量使用

css属性标准值的地方都可以使用变量，标准值被直接替换为变量名，格式为“$变量名”，代码如下： 

//sass variables
$font-family:microsoft yahei,sans-serif;
$font-size:20px;
body{
    font:$font-size $font-family;
}

在声明变量时，变量值也可以引用其他变量，但是要注意两个变量之间的关系。代码如下： 

$highlight-color: #F90;
$highlight-border: 1px solid $highlight-color;
.selected {
  border: $highlight-border;
}

一般我们定义的变量都为属性值，可直接使用，但是如果变量作为属性或在某些特殊情况下等则必须要以#{$variables}形式使用。代码如下： 

$side:left;
.title{
    border: 1px solid black;
    //变量在CSS属性中使用
    border-#{$side}-radius:5px;
}

c.全局变量和局部变量

与CSS属性不同，变量可以在css规则块定义之外存在。当变量定义在css规则块内或者它们出现在任何形式的{...}块中（如@media或者@font-face块），那么该变量只能在此规则块内使用，这样的变量叫做局部变量；否则称为全局变量。代码如下： 

$nav-color: #F90;//全局变量，任何地方都可以使用
nav {
  $width: 100px;//局部变量，只能在nav内部使用
  width: $width;
  color: $nav-color;
}

d.默认变量

sass的默认变量仅需要在值后面加上!default即可。 

//sass style
$baseLineHeight:1.5 !default;
body{
    line-height: $baseLineHeight; 
}

//css style
body{
    line-height:1.5;
}

sass的默认变量一般是用来设置默认值，然后根据需求来覆盖的，覆盖的方式也很简单，只需要在默认变量之前重新声明下变量即可。 

//sass style
$baseLineHeight:        2;
$baseLineHeight:        1.5 !default;
body{
    line-height: $baseLineHeight; 
}

//css style
body{
    line-height:2;
}

可以看出现在编译后的line-height为2，而不是我们默认的1.5。默认变量的价值在进行组件化开发的时候会非常有用。

e.示例 

//sass variables
$font-family:microsoft yahei,sans-serif;
$font-size:20px;
$primary-color:blue;
$margin-top:10px;
body{
    font:$font-size $font-family;
    color:$primary-color;
}
$side:left;
.title{
    border: 1px solid black;
    //变量在CSS属性中使用
    border-#{$side}-radius:5px;
}
.mt{
    margin-top: $margin-top; 
}
//编译后的CSS代码如下：
body {
font: 20px microsoft yahei, sans-serif;
color: blue;
}
.title {
border: 1px solid black;
border-left-radius: 5px;
}
.mt {
margin-top: 10px;
}

②嵌套

sass的嵌套包括两种：一种是选择器的嵌套；另一种是属性的嵌套。我们一般说起或用到的都是选择器的嵌套。

a.选择器的嵌套

所谓选择器嵌套指的是在一个选择器中嵌套另一个选择器来实现继承，从而增强了sass文件的结构性和可读性。代码如下： 

//sass nesting
nav{
    ul{
        list-style-type: none;
    }
    li{
        display: inline-block;
    }
    a{
        display: block;
        padding: 5px 10px;
        text-decoration: none;
        background-color: black;
    }
    a:hover{
        color: white;
    }
}
//编译后的CSS代码如下：
nav ul {
list-style-type: none;
}
nav li {
display: inline-block;
}
nav a {
display: block;
padding: 5px 10px;
text-decoration: none;
background-color: black;
}
nav a:hover {
color: white;
}

在选择器嵌套中，可以使用父元素选择器标识符“&”。比如a:hover伪类，代码如下： 

a {
    display: block;
    padding: 0 10px;
    color: #fff;
　　&:hover { 
color: #ffb3ff;
    }
}

b.属性的嵌套

所谓属性嵌套指的是有些属性拥有同一个开始单词，如border-width，border-color都是以border开头。比如border-color属性，可以写成： 

p {
　　border: {
　　　　color: red;
　　}
}

注意：border后面必须加上冒号

③@import导入SASS文件

sass的导入(@import)规则和CSS的有所不同，编译时会将@import的scss文件合并进来只生成一个CSS文件。但是如果你在sass文件中导入css文件如@import 'reset.css'，那效果跟普通CSS导入样式文件一样，导入的css文件不会合并到编译后的文件中，而是以@import方式存在。

使用sass的@import导入文件时不需要指明文件全名，可以省略.sass或.scss文件后缀。

sass局部文件有个特殊的命名约定，此约定为：sass局部文件的文件名以下划线开头，如：“_partials.scss”。局部文件在导入时可以不写下划线，可写成@import "partials"。

被导入的sass文件_resetcss.scss 

//_resetcss.scss
html,
body,
ul,
ol {
   margin: 0;
  padding: 0;
}

需要导入文件的sass文件test.scss 

//sass import other scss files
@import 'resetcss';
//sass variables
$font-family:microsoft yahei,sans-serif;
$font-size:20px;
$primary-color:blue;
$margin-top:10px;
body{
    font:$font-size $font-family;
    color:$primary-color;
}
$side:left;
.title{
    border: 1px solid black;
    border-#{$side}-radius:5px;
}
.mt{
    margin-top: $margin-top; 
}

test.scss编译后的css代码 

html,
body,
ul,
ol {
  margin: 0;
  padding: 0; }
body {
  font: 20px microsoft yahei, sans-serif;
  color: blue; }
.title {
  border: 1px solid black;
  border-left-radius: 5px; }
.mt{
  margin-top: 10px; }

④混合器@mixin

混合器使用@mixin标识符定义。这个标识符给一大段样式赋予一个名字，这样你就可以轻易地通过引用这个名字重用这段样式。混合器可以传递参数，参数名以$符号开始，多个参数以逗号分开，也可以给参数设置默认值。声明的@mixin通过@include来调用。

a.@minix定义混合器

定义一个带有参数的混合器，其目的是实现跨浏览器兼容性的圆角边框，代码如下： 

@mixin border-radius($radius){
    -webkit-border-radius: $radius;
    -moz-border-radius: $radius;
     -ms-border-radius: $radius;
          border-radius: $radius;
}

b.@include调用混合器

通过@include来使用这个混合器，放在你希望的任何地方。@include调用会把混合器中的所有样式提取出来放在@include被调用的地方。代码如下： 

//use my minix border-radius starting with @include
.circle{
    width: 100px;
    height: 100px;
    background-color: greenyellow;
    @include border-radius(50px);
}
//编译后的css代码
.circle {
  width: 100px;
  height: 100px;
  background-color: greenyellow;
  -webkit-border-radius: 50px;
  -moz-border-radius: 50px;
  -ms-border-radius: 50px;
  border-radius: 50px; }

c.示例

无参数mixin 

//sass style
@mixin center-block {
    margin-left:auto;
    margin-right:auto;
}
.demo{
    @include center-block;
}
//css style
.demo{
    margin-left:auto;
    margin-right:auto;
}

有参数mixin，并设置默认值 

//sass style
@mixin opacity($opacity:50) {
  opacity: $opacity / 100;
  filter: alpha(opacity=$opacity);
}
//css style
.opacity{
  @include opacity; //参数使用默认值
}
.opacity-80{
  @include opacity(80); //传递参数
}

多个参数mixin，并设置默认值

调用时可直接传入值，如果@include传入参数的个数小于@mixin定义参数的个数，则按照顺序表示，后面不足的使用默认值；如果不足的没有默认值则报错。除此之外，还可以选择性的传入参数，使用参数名与值同时传入。 

//sass style  
@mixin horizontal-line($border:1px dashed #ccc, $padding:10px){
    border-bottom:$border;
    padding-top:$padding;
    padding-bottom:$padding;  
}
.imgtext-h li{
    @include horizontal-line(1px solid #ccc);
}
.imgtext-h--product li{
    @include horizontal-line($padding:15px);
}
//css style
.imgtext-h li {
    border-bottom: 1px solid #cccccc;
    padding-top: 10px;
    padding-bottom: 10px;
}
.imgtext-h--product li {
    border-bottom: 1px dashed #cccccc;
    padding-top: 15px;
    padding-bottom: 15px;
}

⑤选择器继承@extend

sass中，允许一个选择器继承另一个选择器的所有样式，并联合声明。选择器继承使用关键词@extend，后面紧跟需要继承的选择器。circle类选择器继承mt类选择器，代码如下：

mt类选择器样式： 

.mt{
    margin-top: $margin-top; 
}

circle类选择器样式 

.circle{
    @extend .mt;//inherited
    width: 100px;
    height: 100px;
    background-color: greenyellow;
    @include border-radius(50px);
}

编译后的CSS代码： 

.mt, .circle {
  margin-top: 10px; }
.circle {
  width: 100px;
  height: 100px;
  background-color: greenyellow;
  -webkit-border-radius: 50px;
  -moz-border-radius: 50px;
  -ms-border-radius: 50px;
  border-radius: 50px; }

占位选择器%

从sass 3.2.0以后就可以定义占位选择器%。这种选择器的优势在于：如果不调用则不会有任何多余的css文件，避免了以前在一些基础的文件中预定义了很多基础的样式，然后实际应用中不管是否使用了@extend去继承相应的样式，都会解析出来所有的样式。占位选择器以%标识定义，通过@extend调用。 

//sass style
%ir{
  color: transparent;
  text-shadow: none;
  background-color: transparent;
  border: 0;
}
%clearfix{
  @if $lte7 {
    *zoom: 1;
  }
  &:before,
  &:after {
    content: "";
    display: table;
    font: 0/0 a;
  }
  &:after {
    clear: both;
  }
}
#header{
  h1{
    @extend %ir;
    width:300px;
  }
}
.ir{
  @extend %ir;
}
//css style
#header h1,
.ir{
  color: transparent;
  text-shadow: none;
  background-color: transparent;
  border: 0;
}
#header h1{
  width:300px;
}


⑥运算符：+、-、*、/、%

sass允许在代码中使用运算符进行计算，可以对数值型的值(如：数字、颜色、变量等)进行+、-、*、/和%运算。请注意运算符前后请留一个空格，不然会出错。引用官网代码如下： 

.container { width: 100%; }
article[role="main"] {
  float: left;
  width: 600px / 960px * 100%;
}
aside[role="complimentary"] {
  float: right;
  width: 300px / 960px * 100%;
}

编译后的CSS代码 

.container {
  width: 100%;
}
article[role="main"] {
  float: left;
  width: 62.5%;
}
aside[role="complimentary"] {
  float: right;
  width: 31.25%;
}

⑦颜色函数

sass定义了很多函数可供使用，当然你也可以自己定义函数，以@fuction开始。sass的官方函数链接为：http://sass-lang.com/documentation/Sass/Script/Functions.html。实际项目中我们使用最多的应该是颜色函数，而颜色函数中又以lighten减淡和darken加深为最常使用，其调用方法为 

lighten($color,$amount);
darken($color,$amount);

注意：它们的第一个参数都是颜色值，第二个参数都是百分比。

SASS提供了一些内置的颜色函数，以便生成系列颜色。 

　　lighten(#cc3, 10%) // #d6d65c
　　darken(#cc3, 10%) // #a3a329
　　grayscale(#cc3) // #808080
　　complement(#cc3) // #33c

示例代码： 

//sass style                    
$baseFontSize:10px !default;
$gray:#ccc !defualt;        
// pixels to rems，自定义函数
@function pxToRem($px) {
  @return $px / $baseFontSize * 1rem;
}
body{
  font-size:$baseFontSize;
  //颜色函数
  color:lighten($gray,10%);
}
.test{
  font-size:pxToRem(16px);
 //颜色函数
  color:darken($gray,10%);
}
//css style
body{
  font-size:10px;
  color:#E6E6E6;
}
.test{
  font-size:1.6rem;
  color:#B3B3B3;
}

⑧注释

SASS共有两种注释风格：

a.标准的CSS注释 

 /* comment */ 
会保留到编译后的文件。

b.单行注释 

 // comment
只保留在SASS源文件中，编译后被省略。

注意：在/*后面加一个感叹号，表示这是"重要注释"。即使是压缩模式编译，也会保留这行注释，通常可以用于声明版权信息。注释形式如下： 

/*! comment*/
重要注释，编译后人存在。

5、高级用法

①@if条件语句

a.单独使用@if语句 

/*@if directive*/
p{
    margin: 10px auto;
    @if 1+1==2{
        border:1px dashed blue;
    }
    @if 5<3{
     border:2px solid black;
    }
    @if null{
     border:3px solid yellow;   
    }
}

//编译后CSS代码
/*@if directive*/
p {
  margin: 10px auto;
  border: 1px dashed blue; }

b.@if和@else结合使用 

/*@if and @else*/
$type:monster;
p{
    @if $type==ocean{
        color:blue;
    } @else if $type==matador{
        color:red;
    }@else if $type==monster{
        color:green;
    }@else{
        color:black;
    }    
}
//编译后CSS代码
/*@if and @else*/
p {
  color: green; }

②@for循环语句

@for循环语句有两种形式，分别为： 

@for $var from <start> through <end> 
@for $var from <start> to <end>

注意：$var可以为任意变量，如$i；<start>和<end>返回值为整数。两者的区别：from ... through循环包括<end>值；而from ... to循环不包括<end>值。

a.@for $var from <start> through <end>  

// form through:including the value of <end>
@for $i from 3 through 8{
    .para-#{$i}{
        width:2rem * $i;
    }
} 
//编译后CSS代码
/* form through:including the value of <end>*/
.para-3 {
  width: 6rem; }

.para-4 {
  width: 8rem; }

.para-5 {
  width: 10rem; }

.para-6 {
  width: 12rem; }

.para-7 {
  width: 14rem; }

.para-8 {
  width: 16rem; }

b.@for $var from <start> to <end> 

// form to:not including the value of <end>
@for $i from 1 to 3{
    .para-#{$i}{
        width:2rem * $i;
    }
} 
//编译后CSS代码

/* form to:not including the value of <end>*/
.para-1 {
  width: 2rem; }

.para-2 {
  width: 4rem; }

③@while循环语句

sass支持while循环，代码如下： 

/* while diretive*/
$i:8;
@while $i>0{
    .para-#{$i}{
        padding:5px*$i;
        $i:$i - 1;
    }
}

//编译后的CSS代码
/* while diretive*/
.para-8 {
  padding: 40px; }

.para-7 {
  padding: 35px; }

.para-6 {
  padding: 30px; }

.para-5 {
  padding: 25px; }

.para-4 {
  padding: 20px; }

.para-3 {
  padding: 15px; }

.para-2 {
  padding: 10px; }

.para-1 {
  padding: 5px; }

以上while循环语句表示：i>0时，输出选择器样式，并且每循环一次i的值减1。

④@each循环语句

@each语法为： 

@each $var in <list or map>

其中$var可以为任意变量，如：$len、$animal等；而list和map表示list类型数据和map类型数据。

单个字段list数据循环 

/*@each diretive*/
@each $bg in adult,baby,dog,robot,yellowman{
    .#{$bg}-bg{
        background-image: url(../img/#{$bg}.jpg);
    }
}

//编译后的CSS代码
/*@each diretive*/
.adult-bg {
  background-image: url(../img/adult.jpg); }

.baby-bg {
  background-image: url(../img/baby.jpg); }

.dog-bg {
  background-image: url(../img/dog.jpg); }

.robot-bg {
  background-image: url(../img/robot.jpg); }

.yellowman-bg {
  background-image: url(../img/yellowman.jpg); }

多个字段list数据循环 

//sass style
$animal-data: (puma, black, default),(sea-slug, blue, pointer),(egret, white, move);
@each $animal, $color, $cursor in $animal-data {
  .#{$animal}-icon {
    background-image: url('/images/#{$animal}.png');
    border: 2px solid $color;
    cursor: $cursor;
  }
}

//css style
.puma-icon {
  background-image: url('/images/puma.png');
  border: 2px solid black;
  cursor: default; 
}
.sea-slug-icon {
  background-image: url('/images/sea-slug.png');
  border: 2px solid blue;
  cursor: pointer; 
}
.egret-icon {
  background-image: url('/images/egret.png');
  border: 2px solid white;
  cursor: move; 
}

多个字段map数据循环 

//sass style
$headings: (h1: 2em, h2: 1.5em, h3: 1.2em);
@each $header, $size in $headings {
  #{$header} {
    font-size: $size;
  }
}

//css style
h1 {
  font-size: 2em; 
}
h2 {
  font-size: 1.5em; 
}
h3 {
  font-size: 1.2em; 
}

⑤三目判断

语法为： 

if($condition, $if_true, $if_false)

三个参数分别表示：条件，条件为真的值，条件为假的值。 

if(true, 1px, 2px) => 1px
if(false, 1px, 2px) => 2px
